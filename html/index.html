1. IIDAXE-dp

IIDAXE-dp TOP(概要)

BEMANIシリーズの一分野における難易度比較を、
ヒエラルキー形式で考察, 提言, 表現する為のツール。

機能説明

    Generateボタン 及びウィンドウ

  • 上部ヘッダー内Generateボタンにより生成用モーダルウィンドウ表示。
  • VERSIONセレクトボックス変更で
    MUSIC & SCOREセレクトボックス内容が変更。
  • MUSIC & SCOREのセレクトボックスはShiftキーやCtrlキーを
    押しながらで複数選択可能(Allボタンで全選択状態に)。
  • Box Generateボタンにより MUSIC & SCORE選択中の内容で
    ドラッグ可能なボックスを生成。
  • Deleteボタンにより 一括削除用モーダル表示。

    • Deleteボタン 及びウィンドウ

  • 生成済みボックスがSettedセレクトボックス内に表示
    (マルチセレクト可能)。
  • ＞＞ボタンにより選択中の対象をSettedからDeleteへ移動。
  • ＜＜ボタンにより選択中の対象をDeleteからSettedへ移動。
  • DeleteボタンによりDeleteセレクトボックス内の
    ボックス全てを一括削除。

    • Config Searchボタン 及びウィンドウ

  • 配置済みのボックスの位置を検索, 選択しSearchボタンにて
    その位置に表示を移動。
    • (※正しく動作しない可能性有り)
  • Name Saveにて作成者名保存しページタイトルに反映。
  • 作成した内容をTwitter共有用URLを作成し、
    Tweetボタンにて投稿可能(もしくは下部URLを利用)。
  • 共有用URLからIIDAXEを開いた場合、
    URL作成時のウィンドウサイズにて別ウィンドウが開きます。
    (※別ウィンドウがポップアップブロック等により開かない事が有りますので許可してください)

    • ヘッダー内他ボタン

  • Save, Loadボタンにより状態の保存, 読出(local Storage使用)。
    • (保存, 読出を行う内容は、"生成および配置した位置",
    "TARGET LEVEL, OPTIONセレクトボックスの選択情報")
  • Settedボタンにより配置済みの一覧表示、曲名クリックで移動。
    モーダル内Not Settedボタンで未配置一覧表示に切り替え。
  • Initializeボタンで全て削除(local Storage内保存内容も削除)。
  • Informationボタンで当説明と同内容の表示。

    • 軸, 座標について

  • X軸 : 対象譜面において特異であるか
    (左部 特異である[singular], 右部 一般的である[ordinary])。
  • Y軸 : 対象譜面において設定した目標の達成難易度
    (上部 達成難度高[difficult], 下部 達成難度低[easy])。
    • (同じY座標にて、同程度の難易度ではあると思っても
    個人差, 譜面の特異性, 特徴から一般的で無い と考えるならば左部へ)。

    その他

  • 生成したボックス複数を選択する様にドラッグすると
    複数選択状態に(枠線が赤の状態)、まとめてドラッグ可能。
  • Shiftキー+Ctrlキーの同時押しをしながら
    ボックスをダブルクリックで対象の削除。
  • Altキーを押しながらボックスをダブルクリックで
    対象のTextageの譜面ページへ別ウィンドウにて移動。
  • マウスオーバーで吹き出しによる曲名表示。
  • 一部楽曲名が違います(ダブルクォーテーションの関係)

IIDAXE-dp TOP(Overview)

A tool for DP difficult12 in beatmania IIDX
to consider, propose and express difficulty comparison
in a certain field in hierarchical form.

Feature Description

    Generate Button with window

  • By "Generate Button" in header push, display window for genarate.
  • If change the "VERSION" sellectbox, changed contents of "MUSIC & SCORE" sellectbox.
  • Contents of "MUSIC & SCORE" sellectbox can be sellected multiple by Shift or Ctrl(All button can all sellected).
  • "Box Generate" button make draggable BOX with sellected in "MUSIC & SCORE".
  • "Delete" button call window for bulk delete.

    • Delete Button with window

  • Displayed list of made draggable box in "Setted" sellectbox (Multiple sellectable).
  • "＞＞" button move sellected to "Delete" from "Setted".
  • "＜＜" button move sellected to "Setted" from "Delete".
  • "Delete" button pushed, delete draggable boxes of sellected contents in "Delete" sellectbox.

    • Config Search Button with window

  • Search position of setted boxes, display moved to the position by pushing "Search" button.
    • (※May not work properly)
  • "Name Save" button saves author name and applies that name.
  • Created tier chart can be posted by "Tweet" button with URL share( or use direct URL).

  • A separate window will open depending on the window size when creating the URL.
    • (※Since another window may not open due to a pop-up block etc., please allow it)

    Other button in headder

  • "Save" and "Load" buttons can save and load position status(Using local Storage).
    • (Save and load contents is "Generate and set position", "status ofTARGET LEVEl and OPTION sellectbox")
  • By "Setted" button, display the setted List. And "Not Setted" button toggled not setted list.
  • "Initialize" button delete All setted box( with local Storage).
  • Information display this explanatory text.

    • Axis, Position

  • X Axis : Whether an target score is singular(left side : It's singular, right side : It's ordinary).
  • Y Axis : Compleat difficulty of setted on "TARGET LEVEL".(Top side : It's difficult, Buttom side : It's easy).
    • (If you think on same level Y axis both score and it has singularyty, you should set left side)

    Other function

  • Generated boxes can be selected multi drag(Borders are red).
  • If double click the box with pushing "Shift key + Ctrl key", target box deleted.
  • If double click the box with pushing "Alt key", open on textage's score page.
  • By mouse over display music name.
  • Some music name different original name(In relation to double quotation).

Last Update(Added events)

  • ARC SCORE(FINAL)
  • ARENA(6th round)
  • DELABITY LABORATORY(FLOOR 3)
  • The Bibliotheca of Mirage(Last Archive)
  • 20th Anniversary Secret Sound
  • BEMANI Vote selection 2019

D-Posts(概要)

就労移行支援事業所内のRuby on Railsの学習で作製
Twitterと同様の登録性, 文章投稿, 画像投稿, 閲覧機能

機能説明

    新規登録

  • アカウント名, E-mailアドレス, ログイン用パスワードを入力し新規登録完了
  • Log inボタンより登録に用いたE-mailアドレス, パスワードでログイン

    • Homeボタン

  • 自身のアカウント 及びフォローしているアカウントの投稿を
    時系列に沿って表示

    • 投稿

  • "Compose new micropost" より文章を入力し投稿可能(空白不可能, 140字以内)
    画像も添付可能

  • Usersボタン
  • 登録済みのユーザー一覧表示(現在 大半はダミーのデータ)
    ユーザー ページよりフォローが可能。
